# Astro Blog Tutorial

The completed example from [Astro's official "Build a blog" tutorial](https://docs.astro.build/en/tutorial/0-introduction/) — a static-site blog with Markdown posts, tags, and an RSS feed.

## What it does

- Static, Markdown-driven blog with three sample posts.
- Tag pages, "About" page, RSS feed.
- Pure Astro — no client-side JavaScript framework.

## Notes for OS8 users

- A great starting template if you want to fork-and-customize an Astro blog inside OS8.
- Pinned to the `complete` branch's HEAD SHA (the upstream uses staged branches per tutorial chapter; `complete` is the finished demo).
- No outbound network required after install.
